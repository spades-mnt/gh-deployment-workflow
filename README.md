# gh-deployment-workflow

This project demonstrates how to deploy a simple static website to **Github Pages** using **Github Actions**.


## Project Overview


- A basic 'index.html' file is hosted on Github Pages.
- A Github Actions workflow automatically deploys the site.
- Every push to the main branch that changes 'index.html' triggers a redeployment.


## How It Works

1. You push changes to rhe 'main' branch.
2. Github Actions run the workflow.
3. The website is deployed to Github Pages automatically.
